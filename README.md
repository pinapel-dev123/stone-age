# Stone age
a simple sandbox game (minecraft pre-classic or cave game modification)

# How to use
1.Download any Minecraft launcher  
2.Install rd-161348  
3.Goto your minecraft directory  
4.Replace rd-161348.jar with stone-age.jar (If don't work rename it with the same name rd-161348)  
5.Open your Minecraft launcher  
6.Run rd-161348  
7.Done! You can play stone age
